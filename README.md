# JAMMM Together
## Überblick
JAMMM Together ist eine webbasierte Plattform zur zentralen Organisation der Hochzeitsplanung. Sie vereint alle wichtigen Funktionen an einem Ort und ermöglicht eine strukturierte, effiziente und stressfreie Planung.
________________________________________
## Problemstellung
Die Planung einer Hochzeit umfasst viele parallele Aufgaben wie Terminabstimmung, Gästemanagement und Organisation von Dienstleistungen.
Diese Informationen sind oft verteilt, was zu:

  •	*Unübersichtlichkeit*
  
  •	*ineffizienter Planung*
  
  •	*erhöhtem Stress*

führt.
________________________________________
## Lösung
Unsere Plattform bietet eine zentrale Lösung mit:

- To-Do-Liste
- Kalenderfunktion
-	Countdown
-	Catering-Vorschläge
-	Login-System
-	Öffentlicher Informationsbereich 
________________________________________
## Features (technisch)
-	Benutzer-Authentifizierung mit Supabase
-	CRUD-To-Do-System (Create, Read, Update, Delete)
-	Dynamischer Kalender 
-	Echtzeit-Countdown
-	Öffentliche und private Seitenlogik 
________________________________________
## Tech Stack
-	HTML, CSS, JavaScript
-	Bootstrap 
-	Supabase 
-	Visual Studio Code 
-	GitHub 
________________________________________
## Installation und Setup
1. Repository klonen
   - git clone https://github.com/if24b107/ITP-SS26.git
2. Projekt öffnen
   - Das Projekt kann in Visual Studio Code oder einem anderen Editor geöffnet werden.
3. Supabase konfigurieren
4. Projekt starten
   - Die Datei index.html im Browser öffnen.
